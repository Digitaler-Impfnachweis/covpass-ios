Update: What's new?

With this app version we have made optimizations and fixed small bugs.

  • "Check validity" removed


    There are currently no entry rules regarding COVID-19, so we have removed the function to check the validity of certificates in various countries.

  • Yellow information box


    A notice box on the entry page informs you about new features and shutdowns of various services within the app.

  • Contact/Imprint


    The contact details can now be found in the legal notice.

  • Checking rules


    Since various checking rules expire at the end of May, they can no longer be updated manually in the app.
